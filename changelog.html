Aside — Changelog

Every shipped version of the iOS app + the source bundle on Polar. Newest first.

v1.1 Shipped live 2026-05-27

  • App and App Store listing now in 15 languages — English, Vietnamese, Chinese (Simplified + Traditional), Japanese, Korean, Spanish (Spain + Mexico), Portuguese (Brazil), French, German, Italian, Russian, Arabic (with RTL), Indonesian.
  • In-app Language picker in Settings; auto-detect device language on first launch.
  • Five Apple Intelligence demos in the chat:
    • Image Playground in the composer — sparkle button opens Apple's image-generation sheet seeded with the current input.
    • Writing Tools on the input field — long-press → Rewrite / Proofread / Summarize / Make Friendly / etc.
    • Summarize chat — chat menu item, streams a concise 3-sentence summary on-device.
    • Smart Reply chips — 3 short tap-to-fill reply candidates above the keyboard, generated after every assistant response.
    • Translate — long-press any message → Translate → Apple's native on-device translation overlay.
  • Source bundle on Polar regenerated to match.

v1.0 Shipped 2026-05-21

  • Initial App Store release.
  • On-device chat with Apple Intelligence (FoundationModels) — no cloud, no account, no API key.
  • Persona presets + custom system prompt.
  • SwiftData chat history with search, share-as-Markdown, regenerate, delete.
  • Voice input (Speech) with optional auto-send.
  • Photo attachment with on-device OCR + Vision classification — Aside reads any image you send it.
  • Markdown rendering with code-block syntax styling.
  • Source bundle launched at $19 on Polar — SwiftUI + FoundationModels + SwiftData, ~3,500 lines.

Pulse (Paddle revenue dashboard) and Skyline (boarding-pass scanner + 3D flight log) are the other apps in Tony's portfolio. All three are built on the same on-device-first philosophy.
